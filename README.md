## Introdução
Padrão criado em 1976 pela Xerox

### Controller
Executa os requests de entrada na aplicação.

* Verbos HTTP
* Result
* Rotas
* Action Results
* Passagem de parâmetro
* ViewData: Componente de transporte da informação da Controller para a View

### Model 
* DataAnnotations
* Validação de Modelo

### View
* HtmlHelpers
* TagHelpers
* Views de Configuração
* Validação de dados
* Partial Views
* Proteção contra CSRF
* Scaffold
* Razor
* Bunding e Minification
